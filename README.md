# Hotel Booking App – Modern Reservation Platform

This project is a modern hotel booking application built with Flutter.
It provides users with a seamless experience to search, explore, and reserve hotels with an intuitive and responsive interface.

---

## Overview

The application is designed to simplify the hotel reservation process by offering real-time search, detailed hotel information, and a smooth booking workflow.
It follows a scalable architecture and integrates backend services for authentication and data management.

---

## Features

* Search hotels by location, date, and availability
* View detailed hotel information including images, amenities, and pricing
* Browse popular and recommended hotels
* Select dates using an interactive calendar
* Book rooms with a streamlined reservation process
* View and manage booking history
* User authentication and session management

---

## Tech Stack

* Flutter – Cross-platform mobile development
* HTTP Client – API communication


---

## Getting Started

### Prerequisites

* Flutter SDK installed
* Android Studio or VS Code
* Emulator or physical device

### Installation

```bash
git clone
cd hotel-booking-app
flutter pub get
flutter run
```
